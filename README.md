## Usage

```
./guenther [options]
    -j, --jid JID                    Jabber identifier
    -p, --password PASSWORD          Password
    -r, --room ROOM                  Multi-user chat room (room@conference.example.com/nick)
    -d, --debug                      Enables XMPP debug logging
    -h, --help                       Prints this help message
```
